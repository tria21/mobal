-Perkenalan Web Framework
framework adalah kerangka kerja yang digunakan untuk mengembangkan aplikasi berbasis web maupun desktop.
salah satu framework yang digunakan untuk perkuliahan kali ini adalah framework laravel.

-perkenalan laravel
pada pertemuan pertama mata kuliah DPW kali ini dikenalkan dengan salah satu web framework yang dapat digunakan yaitu laravel.
laravel digunakan karena pada saat melakukan praktikum ke depannya mata kuliah ini akan menggunakan bahasa pemrograman PHP.
laravel merupakan salah satu framework yang dapat menunjang kinerja PHP. dengan adanya framework memungkinkan penggunaan bahasa
pemrograman PHP dalam membuat web semakin cepat.

-penginstallan laravel
untuk menginstall laravel kita membutuhkan PHP versi di atas 7.2. menginstall composer, menginstall laravel versi 6.x, menginstall
text editor seperti visual studio code

-pengenalan git
untuk pengumpulan tugas pada mata kuliah ini menggunakan git, oleh karena itu setiap mahasiswa harus menginstall git pada PC masing-masing
selain itu mahasiswa harus memiliki akun github. git adalah salah satu sistem pengontrol pada proyek perangkat lunak. mata kuliah ini
menggunakan git karena cukup mudah untuk digunakan, dan git juga sangat familiar digunakan di dunia pemrograman. selain itu ketika menggunakan
git akan terlihat siapa saja yang berpartisipasi pada suatu projek itu sehingga memudahkan dosen untuk memberikan nilai kepada mahasiswa sesuai
dengan kontribusinya di dala projek tersebut
